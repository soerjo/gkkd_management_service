## Description

Apps for backend of E-gereja front ent

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Migration

```bash
# generate query
$ npm run migration:generate

# run migration
$ npm run migration:run

# revert migration
$ npm run migration:revert

```
